EduCatálogo
Aplicación web tipo catálogo desarrollada como proyecto del módulo de Front End en el Politécnico Grancolombiano. Permite a los usuarios explorar cursos digitales en áreas como tecnología, arte, finanzas y marketing.

¿De qué trata?
EduCatálogo es una plataforma donde los usuarios pueden:

1. Explorar un catálogo de cursos con buscador y filtro por categoría
2. Ver el detalle completo de cada curso
3. Guardar cursos en favoritos (persisten en el navegador)
4. Contactar a la plataforma mediante un formulario
5. Iniciar sesión o registrarse

Tecnologías utilizadas

-HTML5
-CSS3 (variables CSS, Flexbox, Grid, diseño responsivo)
-JavaScript ES6+ (fetch API, localStorage, manipulación del DOM)
-JSON (fuente de datos de los cursos)
-Google Fonts (Montserrat + Open Sans)
